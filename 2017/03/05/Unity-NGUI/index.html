Unity_NGUI

  • NGUI(3-11-1)

unity3D游戏的shader和UI的学习是十分重要的，对此进行强化学习可以更好的设计游戏

NGUI的介绍和导入
创建UIRoot

##Label

###属性：
字体样式：font
NGUI :font(所有制作字体文件)
Unity:动态字体（系统自带）按设置按钮设置，可以在本机电脑字体库进行导入unity->Project中进行引用设置
字体大小：font size 最大
字体文本：Text 回车换行
字体伸缩：Overflow
Shrink Content: 自动适应
Clamp Comtent: 裁剪掉不能显示到的部分
Resize freely: 固定大小
Resize Height: 只能调节宽度，高度随文本变化
对齐方式：Alignment
justfied: 自动扩展（宽度一致） 只对英文
颜色渐变：Gradient
其他：Effect 阴影(Shadow) 边框(Outline)
颜色：Color Tint

##精灵（显示图片）Sprites

###属性
图集：Atlas
图片: Sprite
类型：Type
翻转：Flip
颜色：Color Tint

##面板 panel 相当于一个集合多个控件的界面

###属性
设置透明度：Alpha
设置剪切样式：Clipping

##按钮 Button

###设置
在有点击功能的控件上右键 attach->Box Collider
再右键 attach->Button Scrip

###属性
点击事件：On Click 编写代码设置

###创建图集 Atlas
NGUI->Open->Atlas Maker 点击unity内部文件中的图片进行导入
图集的应用同上面 Sprites

##九宫切图 Sliced
创建Sprite->type->Sliced
按Edit进行编辑

##动态字体的设置 Dynamic
将字体导入unity中的文件夹

NGUI->Open-<Font Maker
属性设置为dynamic
Create the Font即可使用

##Widget属性
pivot 相对位置
depth 深度
size 大小 可以用比例实现 Snap复原 Free 自由设置 BasedOnHeight 高度为基准 高度不可变 另一个宽度同理

##锚点设置 Anchor 实现固定点屏幕上的相对变化
Sprite与Camera大小保持一致
点击Sprite 在Inspector中的Anchors中设置
type Unified(低级设置下面以此为例) Advanced（高级设置）
Exexute 执行
Target 目标
Left
Right
Botton
Top
案例：头像 LT
小地图 RT
技能栏 CB

##动画 Tween(补间动画，差值运算)
创建Label->Tween->Alpha
From 0:不可见 1：完全可见
to 与from进行转换
Tweener
Play Style
Once 一次
loop 从头循环
ping pang 往返循环
Animation Curve 变换曲线
Duration 时长
Start Delay 延时开始
Tween Group 动画组
Ignore TimeScale 实现暂停或慢动画

动画还有Color Width Height Position(位置 如：飘入) Rotation Scale(放大) Transfrom(组件 相当于位置变化)

##滑动器（条） 血条 声音调节大小
Create->Sprite
添加 Box Colider
添加 Slider Script
UISlider
Alpha 指定下面的透明度
steps 步长 移动变化
Foreground 前置背景 可拖动背景 子控件
Background 后置背景
Thumb 游标
Direction 方向
