Unity_Shader

  • Unity shader

有人说，程序员的三大浪漫是编译原理、操作系统和图形学。
我们是程序员中的“外貌协会”，期待着用代码编写出炫丽多姿的世界。这就是我们的浪漫。

#GPU流水线

##几何阶段：
顶点着色器 Vertex Shader
是完全可编程的，他通常用于实现顶点空间变换、顶点着色等功能（输出后续所需要的数据）
曲面细分着色器 Tessellation Shader
是可选的着色器，它用于细分图元。
几何着色器 Geometry Shader
可选着色器，它用于执行逐图元的着色操作，或者被用于产生更多的图元
剪裁 Clipping
可配置的，目的是将那些不在摄像机内的顶点剪裁掉，并提出某些三角图元的面片
屏幕映射 Screen Mapping
这个阶段是不可配置和编程的，他负责把每个图元的坐标转换到屏幕坐标系中

##光栅阶段
三角形设置（Triangle Setup）和三角形遍历阶段(Triangle Traversal) 是固定函数的阶段
片元着色器 Fragment Shader
是完全可编程的，用于实现逐片元的着色操作
逐片元操作 Per-Fragment Operations
负责执行更多重要的操作。如：修改颜色，深度缓冲，进行混合等。是不可编程的，担有很高的可配置性
