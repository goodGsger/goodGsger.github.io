PlayMaker_1.7.7.1

  • Unity 5.5

Playmaker由第三方软件商Hutong Games开发完成。它的Logo是一个中文的“玩”字，独特、有创意。

Playmaker既是一个可视化脚本工具，又是一个分层逻辑框架。设计师、程序员使用Playmaker能够很快地完成游戏原型制作，既适合独立开发者，又适用于团队协作。Playmaker能够大大提高开发效率，让你的游戏变得更加完美。

要求：
Unity 基础要求
打开PlayMaker的Editior 按照自己的习惯布局
接下来以案例形式学习PlayMaker的应用

Light switch灯开关案例

(1)首先创建场景lightSwitch 包括 cube plane 物体
(2)向cube添加Add FSM (状态机)
(3)start -> State1(默认状态) 可更改 如Close 右键可以另外添加状态
(4)添加事件Event 在Editior中事件添加 TurnOff TurnOn
(5)添加过渡 Add transition->TurnOn
(6)运行后会自动创建一个PlayMakerGUI(主要用于绘制提示性UI)
(7)动作浏览器 GUI -> GUI Button 并对属性进行修改
(8)实现控制灯光 添加灯光
(9)对灯光暗度进行设置 在Close下 动作浏览器中添加 Set light Intensity 设置属性

box open and off 箱子的开和关

(1)导入资源 将箱子的模型放入到场景中保存场景Box
(2)添加FSM 添加状态
(3)添加事件 Close Open
(4)添加按钮 设置属性
(5)关联动画 Play Animation 设置属性 Anim Name
由于上述方法会有先执行动画的弊端 所以 要再添加两个State 进行过渡
(6) 添加 Colsing Opening 进行过渡 用FINISHED 进行跳转
