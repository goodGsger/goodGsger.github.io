Shader之数学

  • Unity shader

不懂数学这不得入内。 ————古希腊柏拉图学院门口的碑文

笛卡尔系坐标

二维笛卡尔系坐标
一个特殊的位置，即原点，它是整个坐标系的中心
两条过原点的互相垂直的矢量，即x轴和y轴。被称为该坐标系的基矢量
OpenGL:左下角（0，0）
DirectX:左上角（0，0）

三维笛卡尔系坐标
三个过原点个相互垂直的轴组成的坐标系
三个轴成为该坐标系的基矢量（basis vector）
if(length == 1)
标准正交基（orthonormal basis）
else
正交基（orthogonal basis）
由于方向的不同 分为左手坐标系和右手坐标系，Unity中观察空间使用的是右手坐标系，模型空间使用的是左手坐标
观察空间：以摄像机为原点的坐标系。在这个坐标系中，摄像机的前向是z轴的负方向

点和矢量

矢量被用于表示相对于某点的偏移量，只有模和方向两个属性。
点作为标量，没有大小之分，只是空间中的一个位置。
虽然两者有相同的表达方式，代表的意义却不同。

矢量的运算

1.矢量和标量的乘法/除法
2.矢量的加法和减法(三角形定则，平行四边形定则)
3.矢量的模 |v| = sqrt(v1^2+v2^2+v3^2)
4.单位矢量 v^ = v/|v| 也被称为被归一化的矢量 对于非零矢量转化此过程称为归一化
零矢量 v = (0,0,0)
5.矢量的点积（内积） dot(a,b)
公式一：a.b = (ax,ay,az).(bx,by,bz) = axbx + ayby + azbz
满足交换律 a.b = b.a
几何意义：投影
性质一：(ka).b = a.(kb) = k(a.b)
性质二：a.(b+c) = a.b + a.c
性质三：v.v = |v|^2
公式二：a.b = |a||b|cos0
6.矢量的叉积（外积）
公式一：a b = (ax,ay,az) (bx,by,bz) = (aybz - azby,azbx - axbz,axby - aybx)
不满足交换律 a b != b a
几何意义：同时垂直a,b的矢量
公式二：|a * b| = |a||b|sin0
几何意义：平行四边形面积
应用：计算垂直于一个平面三角形的矢量；还可以用于判断三角片面的朝向。
坐标系的选择只会影响视觉效果，但不会影响叉积的结果。

矩阵 matrix（母体）黑客帝国的英文名字 The Matrix

将矢量与矩阵联系起来可以更好的研究空间变换

矩阵运算

1.矩阵和标量的乘法
2.矩阵和矩阵的乘法 行乘列 Cij = ai1b1j + ai2b2j + … + ainbnj
性质一：不满足交换律
性质二：满足结合律

特殊的矩阵

1.方块矩阵，简称方阵 ：行和列数目相等的矩阵
对角元素 即m11,m22… 行号 = 列号
对角矩阵 除对角元素外的所有元素都为零的矩阵
2.单位矩阵 In 任何矩阵和它相乘结果还是原来的矩阵 MI = IM = M
3.转置矩阵 对于r c 的矩阵转置成 c r
性质一：矩阵转置的转置等于原矩阵 （M^T）^T = M
性质二：矩阵串接的转置等于反向串接各个矩阵的转置 (AB)^T = B^T A^T
4.逆矩阵
条件：该矩阵是方阵
M^-1 M = I
如果一个矩阵有逆矩阵则说这个矩阵是可逆的或是说非奇异的，反之，说是不可逆的或者说奇异的
性质一：逆矩阵的逆矩阵是原矩阵本身
性质二：单位矩阵的逆矩阵是它本身
性质三：转置矩阵的逆矩阵是逆矩阵的转置
性质四：矩阵串接的逆矩阵等于反向串接各个矩阵的逆矩阵
应用：还原原向量
5.正交矩阵
如果一个方阵M和他的转置矩阵的乘积是单位矩阵，就说这个矩阵是正交的。
M M^T = I
如果一个矩阵是正交的，那么他的转置矩阵和逆矩阵是一样的 M^T = M^-1

矩阵的几何意义：变换

1.线性变换 保留矢量加和矢量乘的变换 如，旋转，缩放，错切，镜像和正交投影
2.仿射变换 就是合并线性变换和平移变化的变换类型 扩展到四维空间 这就是齐次坐标空间

齐次坐标和分解基础变换矩阵（表示纯平移，纯旋转和纯缩放的矩阵）
