3.1type_traits——类型萃取

type_traits 提供了丰富的编译期计算，查询，判断，转换和选择的帮助类
圈复杂度度量：从函数第一行开始，一直往下看程序
遇到以下关键字或者同类的词 +1
if,while,repeat,for,and,or
向case语句中的每一种情况 +1

3.1.1 基本的type_traits

1.简单的type_traits
std::integral_constant:定义编译期常量
std::ture_type,std::false_type 分别实现编译期的true，false
2.类型判断的type_traits
is_integral:判断是否是整形类型
更多查看
一些用法：std::is_const::value
3.判断两个类型之间的关系
a.is_name:判断两个类型是否相同。
用法:std::is_name::value
b.is_base_of:编译期判断两个类型是否是继承关系
用法:std::is_base_of::value A为B的基类 true
c.is_convertible:用来判断前两面的模板参数类型能否转换为后面的的模板参数类型
用法:std::is_convertible::value;
4.类型的转换traits
a.根据模板参数类创建对象时，注意移除引用。
typedef typename std::remove_reference::type U;
b.智能指针引用 创建移除引用 获取再添加
c.cv限定符 std::decay::type
可以移除cv限定符和引用，还可以用于数组和函数
对于数组是移除顶层维度
typedef std::decay::type A;//int
对于函数是添加指针
typedef std::decay::type B;//int() (int)

3.2可变参数模板

用户声明可变参数模板时需要在typename或class后面带上…
作用：a.声明一个参数包，这个参数包可以包含0到任意个数的模板参数。
b.在模板定义的右边，可以将参数包展开成一个一个的参数

3.2.1可变参数模板函数

定义：template
void f(T… args)
{
cout << sizeof…(args) << endl; //打印变参的个数
}
递归方式
逗号表达式和初始化列表方式展开参数包

3.2.2可变参数模板类

std::tupl是一个可变参数模板类
a.模板递归和特化方式展开参数包
//前向声明
template < typename… Args> struct Sum;
//递归模板,并设置参数不能为零,即定义First.
template < typename First, typename… Rest>
struct Sum
{
enum {value = Sum::value + Sum::value};
};
//递归终止类
template < typename Last> struct Sum
{
enum {value = sizeof(Last)};
};

注:可以使用integral_constant消除enum。

b.继承方式展开函数包
//整形序列的定义
template < int…>
struct IndexSeq{};

//继承方式展开参数包
template < int N, int… Indexes>
struct MakeIndexs : MakeIndexs {};// 从小到大排序

//模板特化，终止展开参数包的条件
template < int… Indexes>
struct MakeIndexes<0, indexes...="">
{
typedef IndexSeq type;
};
